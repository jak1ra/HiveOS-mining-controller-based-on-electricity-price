Automatically stops and starts HiveOS miners based on spot electricity prices in Eastern Norway. 

How it works:

Checks the current spot price every 5 minutes using hvakosterstrommen.no. If the price goes above the value you put, it stops the miner, and starts it again when the price drops back down.

Requirements:

Python 3.10+,
A HiveOS account
